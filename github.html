How to make github pages using git system

1 Login to GitHub

If you don't have github account, make a new one.

2 Make a new repository

Using format name yourname.github.io.

3 Git bash

Find your folder that containing the website files, then right-click and choose 'Git Bash Here'.

4 Initiate Local Repository

Type 'git init' on the git bash and then press enter.

5 Enter files to staging area

Type 'git add .' on the git bash and then press enter.

6 Commit Files

Type 'git commit -m "first commit"' on the git bash and then press enter.

7 Remote GitHub Repository

Type 'git remote add origin https://github.com/yourname/yourname.github.io' on the git bash and then press enter.

8 Push File to github repository

Type 'git push origin master' on the git bash and then press enter.

9 Check github Pages

Check whether the files that we pushed has been successful or not, then type your github repository on your browser to open your github pages.
